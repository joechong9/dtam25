📋 Field Scanning Workflow
① Collect
Participant fills in printed OMR answer sheet with a black pen or pencil.
② Scan / Capture
Use flatbed scanner (≥200 DPI) or camera on flat surface with even lighting.
③ Process
Open OMR Scanner → upload pages → click ⚡ Process → review overlay → fix any misreads.
④ Export & Import
Export JSON from scanner → return here → click 📥 Import JSON Results → data appears in log.
⑤ Batch Export
Once all sessions imported, use ⬇️ Export CSV to download combined dataset for SPSS/R.
