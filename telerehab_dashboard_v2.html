Scanning Workflow

Phase 4 OMR data collection pipeline
  1. Participant completes printed OMR answer sheet
  2. Researcher uploads scan (📁) or photographs with camera (📷)
  3. OpenCV auto-corrects perspective for camera photos
  4. Click ⚡ Process — all bubble positions sampled
  5. Review overlay: green = filled · orange = ambiguous · grey = empty
  6. Fix any misreads via the Fix dropdown per item
  7. Export CSV → warehouse · JSON → ML pipeline

Scan Quality Tips

Follow these to maximise auto-detection accuracy
📠 Flatbed Scanner
Use ≥200 DPI · JPEG or PNG · Place sheet flat with no creases · Best accuracy
📷 Camera / Phone
Place sheet on dark flat surface · Even lighting · No shadows · Enable Auto perspective-correct
🎯 If Grid Misaligns
Open Calibration panel · Adjust ΔY per section · Re-process · Save offsets for that printer
